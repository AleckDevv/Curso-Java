## 🚀 Como Executar

### Pré-requisitos

- [Java JDK 8+](https://www.oracle.com/java/technologies/downloads/) instalado na máquina
- Uma IDE de sua preferência (IntelliJ IDEA, Eclipse, VS Code)

### Passo a passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/AleckDevv/Curso-Java.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd Curso-Java
   ```

3. **Abra o projeto na sua IDE** e execute os arquivos `.java` dentro da pasta `src/`.

4. **Ou compile e execute via terminal:**
   ```bash
   javac src/NomeDoArquivo.java
   java src/NomeDoArquivo
   ```
